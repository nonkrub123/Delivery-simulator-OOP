# Delivery simulator OOP
This project about using OOP class to simulate ordering and delivery online. We use Class to manipulate delivery action
## Feature
- ordering
- assigning driver
- placing order
- status track of the delivery
- summary

## How to run
You could run by opening in complier like vscode and run main.py
