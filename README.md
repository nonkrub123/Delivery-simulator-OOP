# Delivery simulator OOP
This project about using OOP class to simulate simple ordering and delivery online. We use Class to manipulate delivery process like placing an order, assigning driver or even delivery.

## Feature
- placing order
- assigning driver
- delivery
- status track of the delivery
- summary

## How to run
You could run by opening in complier like vscode and run main.py
